# MURF Falcon Monthly Buildathon | March Edition

## Overview

Welcome to the MURF Falcon Monthly Buildathon for March! This project is part of our monthly buildathon series where we innovate and build exciting new features.

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn
- Git

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd MURF-30
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm start
   ```
